3. 何謂教學的層次？

1. 上主之師並無固定的教學層次。2開始時，每個教學場景都會牽涉不同的關係，但它們的終極目標一向並無差別；亦即，將此關係轉為神聖關係，以便雙方都能看見聖子的純潔無罪。3沒有誰不是上主之師的學習對象，因此也沒有誰不是他的教學對象。4然而，從實際的面向上看，他無法遇見每一個人，反之亦然。5為此，這一計劃便包含了每一位上主之師將會遇上的具體人事境相。6救恩之中並無意外。7會遇上的人就會遇上，因為二者的聚合能有活出神聖關係的潛力。8他們已準備好彼此相遇。

2. 最單純的教學層次乍看之下相當表淺。2它所蘊含的會晤看似完全隨機；顯然是陌生的倆人在電梯中「偶然」相遇，一個小孩莽莽撞撞的「恰好」擋住了大人的去路，兩名學生「碰巧」一起走路回家。3這些會晤都不是隨機的。4每一個都有潛力成為教與學的場景。5或許，電梯中看似陌生的倆人會彼此微笑；或許，這個大人沒有因為小孩撞到自己而責備他；或許，這兩名學生會成為朋友。6即便是就完全隨機的會晤來說，雙方都有可能忽視分別的利益，即便只是片刻光景。7那片刻光景即已足夠。8救恩已經來臨。

3. 你很難明白，不論是在實相裡或時間之中，這部普世的課程需有教學層次乃是一不具意義的概念。2你對一方的錯覺能使你對另一方也產生錯覺。3在時間裏，上主之師好似是在一個決定下便開始改變自己對世界的看法，然後隨著自己的教導在這新的方向上愈學愈多。4我們已談過時間的錯覺，但教學層次的錯覺卻好似是個不同的東西。5或許，要和你解釋這些層次不可能存在，最好的方法就是直接告訴你下面這點：任何教與學的場景都是上主救贖計劃中的一部分，而祂的計劃並無層次之分，因為那反映的是祂的旨意。6救恩早已存在並準備就緒，且一向如此。7上主之師會在不同的層次上運作，但結果並無二致。

4. 牽涉在教與學場景中的每一個人都會在那段時間內從彼此身上盡其所能的學習，從這層面上說，每一個場景都已是最大限度。2也只有從這層面上說，我們才能夠談論教與學的層次。3一旦以此方式運用這一說法，教學的第二層次則是較為持久的一段關係，其中，雙方將在一段時間內進入一頗為緊湊的教學場景，然後看似分道揚鑣。4一如頭一個層次，這些會晤都不是隨機的，看似結束的關係也並未真正結束。5同樣的，此時的雙方都已在學習上盡了全力。6不過，曾經會晤的則未來必將重逢，因為神聖性即是一切關係的宿命。7上主絕不會錯認自己的聖子。

5. 有些關係一經締結便會延續終身，而第三層次的教學便由此展開。2在這樣的教學場景中，上天將會選定並賜予每個人一位學伴，以便為他提供無窮無盡的學習機會。3一般而言，這樣的關係屬於少數，因為它們的存在代表了牽涉其中的人已同時達致了教與學的平衡，且這樣的平衡確實也已完美。4這並不表示他們一定能認清這點；事實上，他們往往無法認清。5他們甚至會在一段時間內對彼此產生嚴重的敵意，而這敵意也可能延續終身。6然而，只要他們選擇學習，那完美的課程就會出現，而他們也能學得會。7而一旦他們決意學習這一課程，便能拯救那些步履蹣跚或甚至瀕臨失敗的老師。8只要是上主之師，就絕不會找不著他所需的天助。
